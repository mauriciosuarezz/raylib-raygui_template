This is a raylib + raygui template that works with mingw64 compiler and cmake

I made this for a easy way to develop projects with raylib and raygui in C++

Just ensure to have all the cmake tools and the right compiler installed

Put into "src" all your .c , .cpp , .h and .hpp files

Type cmake --build build to compile the project

change the .exe name into the cmakelist.txt "project( *HERE* LANGUAGES C CXX )"

Enjoy :P
